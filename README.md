# ESG Cloud Transformation & Responsible AI Project

## Overview
This repository contains the supporting evidence and experiment logs for the ESG Cloud Transformation and Responsible AI assignment. The project explores the application of cloud computing, conversational AI, and large language models to improve Environmental, Social and Governance (ESG) operations.

## Repository Contents
- Q1: Dialogflow CX ESG Sustainability Chatbot design and implementation.
- Q3: LLM-based ESG message triage prompt design and JSON output examples.
- Q3: Comparison between LLM classification and rule-based approaches.
- Q4: NotebookLM evaluation and academic use case analysis.
- Experiment logs and supporting screenshots.

## Technologies Used
- Google Dialogflow CX
- Google NotebookLM
- Azure OpenAI (Conceptual Architecture)
- GitHub
- Cloud Computing & ESG Frameworks
