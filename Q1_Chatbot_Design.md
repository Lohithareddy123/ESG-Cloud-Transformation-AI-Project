# Q1 - ESG Sustainability Chatbot

## Overview
The chatbot was developed using Google Dialogflow CX to assist employees in reporting sustainability-related incidents.

## Selected Use Case
- Sustainability Incident Reporting Assistant.

## Key Features
- ESG incident reporting.
- Automatic classification of incidents.
- Location and severity capture.
- Escalation of critical incidents.
- User-friendly conversational interface.

## Conversation Flow
1. User starts the chatbot.
2. User reports a water leak.
3. Chatbot captures location.
4. Chatbot asks for severity.
5. Incident is classified.
6. Critical issues are escalated.
7. Confirmation message is displayed.

## Accessibility Considerations
- Simple language.
- Step-by-step guidance.
- Mobile-friendly interface.
- Compatible with assistive technologies.
