# Q3 - Baseline Comparison

## LLM vs Rule-Based Classification

| Feature | LLM | Rule-Based |
|----------|-----|-----------|
| Context Understanding | High | Limited |
| Keyword Matching | Yes | Yes |
| Flexibility | High | Low |
| Accuracy on Complex Inputs | Better | Moderate |
| Need for Human Review | Recommended | Recommended |

## Findings
The LLM demonstrated stronger contextual understanding and produced structured JSON outputs. Rule-based methods were more predictable but lacked flexibility for ambiguous messages.
